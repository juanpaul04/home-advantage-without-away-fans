# Impacto de la Eliminación del Público Visitante en el Fútbol Argentino

## 📌 Descripción

En este proyecto con fines educativos se analiza el impacto en los resultados del fútbol argentino luego de la prohibición del público visitante.  El objetivo fue determinar si la ausencia de hinchada visitante le proporciona una ventaja al equipo local.

---

## 📊 Metodología

- Recolección de datos mediante web scraping de fuentes públicas y datasets públicos de Kaggle
- Limpieza y procesamiento de datos
- Análisis exploratorio y visualización de datos
- Modelado predictivo mediante un clasificador

---

## ⚠️ Aclaración sobre los datos

Parte de los datos utilizados en el proyecto fueron recolectados mediante web scraping desde Transfermarkt.  Dado que el proyecto fue realizado con fines académicos, no se incluyen dichos datasets completos. Se recomienda respetar los términos de uso de las fuentes originales.

---

## 🛠️ Tecnologías utilizadas

- Python
- Pandas
- Numpy
- Matplotlib / Seaborn
- Scikit-learn

---

## 📈 Resultados principales

- Se observó un **efecto inmediato a favor de los equipos locales** tras la eliminación del público visitante.
- Con el paso del tiempo, esta ventaja se **normalizó**, mostrando que el efecto no se mantuvo a largo plazo.
- El análisis indica que, aunque la ausencia de la hinchada visitante generó un impacto en los resultados durante un breve período, otros factores (económicos, organización del torneo)  también influyen y suelen tener más peso  en los resultados.
